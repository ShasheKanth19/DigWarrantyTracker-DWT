# DigWarrantyTracker-DWT
A full-stack web application designed to efficiently manage product warranties, automate expiry tracking, and provide predictive insights for better decision-making.
Digital Warranty Tracker enables users to securely store product purchase details, monitor warranty periods, and receive automated alerts before expiry. The system leverages predictive analytics to forecast warranty renewals, helping users take timely actions and avoid service disruptions.

🛠️ Tech Stack
Backend: Java, Spring Boot, Spring Data JPA
Frontend: HTML, CSS, Bootstrap (responsive UI)
Database: MySQL
Tools & Deployment: Docker, Cloud-based deployment
 Key Features
 Warranty Management: Store and manage product details with warranty timelines
 Automated Alerts: Get notifications for upcoming warranty expirations
 Predictive Analytics: KNN-based model to forecast warranty renewals
 Secure Data Handling: Efficient backend architecture to manage user data reliably
 Responsive Design: Accessible across devices with a clean UI
 Core Functionality
Implements K-Nearest Neighbors (KNN) algorithm to analyze user/product data and predict renewal likelihood
Uses Spring Boot REST APIs for seamless frontend-backend communication
Ensures scalable data operations through Spring Data JPA & MySQL integration
Containerized using Docker for consistent deployment across environments
Objective

To simplify warranty tracking and enhance user experience by combining automation, predictive analytics, and a robust full-stack architecture.
